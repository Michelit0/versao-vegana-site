# Versao Vegana Site

Site público oficial da Versao Vegana, separado do sistema interno.

## Scripts

- `npm run dev`: ambiente local.
- `npm run typecheck`: valida TypeScript.
- `npm run build`: gera build de produção.

## Rotas de produção

- `/`: site oficial.
- `/sistema`: rewrite para o sistema interno hospedado separadamente.

## Conteúdo editavel

Textos, WhatsApp e listas principais ficam em `src/content/siteContent.ts`.
